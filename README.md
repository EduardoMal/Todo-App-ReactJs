Todo App
A simple todo list app with user authentication and task management features.

Features- Add, edit, delete, and mark tasks as completed

- View task details
- User authentication: sign up, sign in, change password, update profile
- Notifications for due tasks
- Responsive design

Tech Stack- React

- React Context API with useContext hook
- useReducer, useState, useRef hooks
- React Router DOM for routing
- Firebase for authentication and Realtime Database

Setup

- 1. Clone the repo
- 2. Install dependencies: npm install
- 3. Create a Firebase project and configure Realtime Database and Authentication
- 4. Update Firebase config in src/utils/config/config.js
- 5. Start the app: npm start

Usage- Sign up or sign in to access tasks

- Add new tasks, edit, delete, or mark as completed
- View task details and update due dates
- Receive notifications for due tasks
